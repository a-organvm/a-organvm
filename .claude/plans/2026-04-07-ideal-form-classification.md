# Ideal Form Classification — What Each Repo Reaches Toward

## Context

Every repo is a thesis — an attempt to instantiate an ideal form. The repos are already-reduced forms; the originating voice (transcripts, prompts) is upstream. This phase classifies the reductions. A subsequent phase traces each ideal back to its voice origin.

Method: classify by what the repo is TRYING TO BE, not by what it contains or whether it has code. Group by the irreducible ideal each repo expresses.

## How Many Distinct Ideals?

15 named above. But some may be the same ideal wearing different clothes:

| Candidates for merging | Why they might be one thing |
|---|---|
| I (Self-Reading) + VIII (Governance as Art) | If the governance IS the self-reading, they're the same recursion |
| II (Alchemical) + VI (Performance) | If performance IS transmutation in real-time |
| III (Narrative) + XIII (World-Building) | If world-building IS narrative with physics |
| IV (Knowledge Atoms) + X (Communal Knowledge) | If the atoms ARE the community's shared memory |
| IX (Multi-Agent) + XIV (Substrate) | If the agents ARE the physics |
| XI (Amplification) + XV (Contribution) | If contributing IS distributing |

If all 6 merges hold, 15 ideals → 9 irreducible ideals. If none hold, 15 remains 15.

**This is a question only the operator can answer** — because the ideal forms exist in the originating voice, not in the repos.

## Next Phase: Voice Origin Tracing

Each ideal form has an originating expression — a prompt, a transcript, a moment in a session where the thesis was first spoken. The 1,322 session exports in praxis-perpetua, the post-flood documents, the personal archives — these contain the voice origins.

Phase 2 would:
1. Read the session exports and post-flood documents
2. Find the first utterance of each ideal form
3. Map: voice → ideal → repos
4. Answer: is each ideal genuinely distinct, or are some the same voice saying the same thing in different sessions?

This phase requires reading the 76% of documents that live in praxis-perpetua. It is the archaeological layer beneath the repos.
