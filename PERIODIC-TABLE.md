**Discovered 2026-04-21** — extracted from deep scan of 25+ repos, not prescribed.

## 10 Irreducible Elements

Every operation in the system decomposes into one or more of these.
Nothing below this level exists.

| # | Element | Symbol | Operation | Analogy |
|---|---------|--------|-----------|---------|
| 1 | **INTAKE** | In | Bring signal into the system | Mouth, receptor, input jack |
| 2 | **STORE** | St | Hold signal across time | Memory, disk, archive |
| 3 | **RETRIEVE** | Re | Access what is stored | Query, lookup, recall |
| 4 | **EVALUATE** | Ev | Compare signal against criteria | Judge, test, audit, score |
| 5 | **TRANSFORM** | Tr | Change signal form | Parse, normalize, atomize, compile |
| 6 | **SYNTHESIZE** | Sy | Combine many signals into one | Merge, fuse, compose, generate |
| 7 | **ROUTE** | Ro | Direct signal to destination | Dispatch, classify, backflow |
| 8 | **AUTHORIZE** | Au | Gate: allow or block passage | Policy, law, schema, promotion |
| 9 | **BIND** | Bi | Connect two things | Link, relate, name, wire |
| 10 | **EMIT** | Em | Send signal out of the system | Publish, export, surface, render |

## Compositions (everything else is built from elements)

| Compound | Formula | Description |
|----------|---------|-------------|
| CLASSIFY | Ev(category) | Evaluate against a category taxonomy |
| LEARN | St(exp) + Ev(outcome) + Tr(behavior) | Store experience, evaluate what worked, transform approach |
| GOVERN | Au + Ev + Em(enforcement) | Authorize rules, evaluate compliance, emit enforcement |
| ATOMIZE | Tr(decompose) | Transform by breaking into smallest units |
| FUSE | Sy(merge) | Synthesize by merging entities |
| SENSE | In(continuous) | Intake on a loop |
| RECONCILE | Ev(conflict) + Sy(resolution) | Evaluate conflicts, synthesize resolution |
| SEQUENCE | St + Re + Ro(ordered) | Store order, retrieve by position, route in sequence |
| RECORD | In + St + Bi(time) | Intake event, store it, bind to timestamp |

## The Signal

Every element operates on the same thing: **STATE**.
State is the voltage. It flows through elements. Each element transforms it.

```
STATE_IN → [element] → STATE_OUT
```

The system is a signal processor. The elements are the modules.
The compositions are the patches. The formations are the presets.
